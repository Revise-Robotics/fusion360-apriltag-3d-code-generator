# Fusion 360 AprilTag 3d code generator

## install

- Clone, and add via Fusion 360's Utilises -> Add-Ins
- Alternatively, install "[GitHub to Fusion 360](https://apps.autodesk.com/FUSION/en/Detail/Index?id=789800822168335025)", then run it and pass it this [repo's url](https://github.com/Revise-Robotics/fusion360-apriltag-3d-code-generator).
